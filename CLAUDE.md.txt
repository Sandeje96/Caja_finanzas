Nombre del Proyecto

FINANZAS IA WHATSAPP

Visión del Producto

Construir un asistente financiero personal impulsado por IA cuya interfaz principal sea WhatsApp.

El usuario debe poder registrar gastos, ingresos, comprobantes y realizar consultas financieras simplemente conversando de forma natural.

El sistema debe actuar como un contador personal inteligente que recuerda todas las operaciones y permite consultar información histórica mediante lenguaje natural.

El usuario nunca debería necesitar completar formularios para registrar movimientos.

Propuesta de Valor

No somos una aplicación de gastos.

Somos un asistente financiero conversacional.

El usuario debe sentir que está hablando con alguien que conoce toda su situación financiera.

Ejemplos:

"gasté 15000 en combustible"

"cobré 850000"

"te envío el ticket"

"cuánto gasté en comida este mes"

"buscame la factura de internet"

"estoy gastando más que el mes pasado"

"cuánto dinero tengo disponible"

Principios del Proyecto

Priorizar simplicidad.

Evitar sobreingeniería.

Construir primero el flujo completo.

Optimizar después.

El dashboard es secundario.

La experiencia conversacional es la prioridad.

Prompt Base de IA

La IA debe actuar como un asistente financiero personal.

Debe interpretar lenguaje natural.

Debe ser precisa con montos.

Nunca inventar movimientos.

Nunca asumir datos inexistentes.

Cuando tenga dudas debe solicitar aclaración.

Las respuestas deben ser breves y orientadas a la acción.
